初期設定

Microsoft 365 アカウントと連携して、Outlook ToDo のタスクを
カレンダーへ自動登録・完了時グレー化を行います。

⚙️ Azure アプリの登録が必要です（初回のみ・5分）
  1. Azure Portal → アプリの登録 　→「新規登録」
  2. 名前: 任意（例: TodoCalendarSync）
  3. サポートされているアカウント: 「任意の組織ディレクトリ内のアカウント」または「個人用 Microsoft アカウント」
  4. リダイレクト URI:「シングルページアプリ (SPA)」を選択 → （ページ読み込み後に表示）
  5. 登録後 → 「APIのアクセス許可」→ 以下を追加（委任）: Tasks.Read　Calendars.ReadWrite　MailboxSettings.ReadWrite
  6. 「概要」画面の「アプリケーション（クライアント）ID」をコピーして下に貼り付け
